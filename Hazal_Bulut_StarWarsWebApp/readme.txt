# StarWarsWebApp Project Instructions

## How to Open the StarWarsWebApp Project

To open the `StarWarsWebApp` project, please follow these steps:

1. **Navigate to the Project Directory:**

   First, locate the `StarWarsWebApp` directory in your file system.

   The path should be: Hazal_Bulut_StarWarsWebApp/StarWarsWebApp

2. **Open the Project File:**

Inside the `StarWarsWebApp` directory, you will find a file named `StarWarsWebApp.csproj`. 

Open this file in your preferred Integrated Development Environment (IDE), such as Visual Studio or Visual Studio Code.

3. **Build and Run the Project:**

Once the project file is open in your IDE, you can build and run the project as needed.
